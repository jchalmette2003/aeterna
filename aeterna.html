☁ Cloud Sync — GitHub Gist
Free multi-device sync via GitHub Gist.
Your data is stored as a private Gist in your GitHub account — accessible from any device with your token. No server, no subscription, no data shared with anyone.
Create at: github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → New token
Required scope: gist (only gist, nothing else needed)
Workflow for multi-device use:
1. Set up token once on your main device → click "Push to Cloud"
2. Copy the Gist ID shown after first push
3. On each other device: enter same token + Gist ID → "Pull from Cloud"
4. After each session: "Push to Cloud" to keep sync current
5. On other devices: "Pull from Cloud" to get latest data
🌐 Host on GitHub Pages
Access AETERNA from any device via a permanent link — completely free.

Step 1: Create a GitHub account

Go to github.com and sign up for a free account if you don't have one.

Step 2: Create a new repository

Click the + icon → New repository. Name it aeterna (or any name). Set visibility to Public. Click "Create repository."

Step 3: Upload your file

Click uploading an existing file → drag your aeterna.html file → commit. Then rename it to index.html (click the file → edit → rename at top).

Step 4: Enable GitHub Pages

Repository → Settings → Pages (left sidebar) → Source: Deploy from a branch → Branch: main → folder: / (root) → Save.

Step 5: Access your app

Your app will be live at: https://yourusername.github.io/aeterna
Wait 1–2 minutes for first deployment. Bookmark this URL on every device.

Step 6: Enable Cloud Sync for multi-device data

Since GitHub Pages doesn't store data between users, set up the GitHub Gist sync (☁ Sync button in sidebar) to share your data across devices via your GitHub account.

Updating the app

When you get an updated version of aeterna.html: go to your repository → click index.html → click the pencil (edit) → paste new content → commit. Changes go live in ~1 minute.
